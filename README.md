# Fish Time Pro

Fish Time 扩展的个人定制版。一个用于 VS Code 的轻量插件：在状态栏实时显示今日工资，悬浮提示下班倒计时；结合中国节假日与自定义休息日，计算本月工作日与累计工资。

## 主要功能
- 状态栏显示今日工资与连续进度百分比
- 悬浮提示工作状态与上下班倒计时
- 统计本月工作天数与累计工资
- 自动识别中国节假日与补班/调休

## 仓库与开发
- 仓库：https://github.com/superior-josh/fish-time-pro.git
- 开发：
    ```shell
    npm install -g @vscode/vsce
    cd fish-time-pro
    npm install
    vsce package
    ```
